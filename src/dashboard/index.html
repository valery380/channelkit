Get a Stable Public URL

Use a free Cloudflare Tunnel to keep the same URL across restarts.

1
Create a Cloudflare account
Go to one.dash.cloudflare.com and sign up for free. No credit card required.
2
Add your domain to Cloudflare
In the Cloudflare dashboard click Add a site, enter your domain name, and choose the Free plan. Cloudflare will show you two nameservers (e.g. ada.ns.cloudflare.com). Go to your domain registrar (GoDaddy, Namecheap, etc.) and replace the existing nameservers with these two. DNS propagation takes a few minutes to a few hours.
3
Create a tunnel
In the sidebar go to Zero Trust → Networks → Connectors, then click Create a tunnel. Choose Cloudflared as the connector, give it a name (e.g. channelkit), and save.
4
Copy the tunnel token
After saving, Cloudflare shows install instructions. Copy the long token that appears after --token. It starts with eyJh…
5
Add a public hostname
In the tunnel settings go to the Public Hostnames tab. Add a hostname on your domain (e.g. channelkit.yourdomain.com) and point it to localhost:4000 with HTTP service type. Save.
6
Paste token & hostname here
Click Set up stable URL below the public URL bar, paste the token and your hostname, then click Save. Click Externalize to connect.

The tunnel stays active as long as ChannelKit is running. If you don't have a domain yet, any cheap registrar works — just point the nameservers to Cloudflare after registering.

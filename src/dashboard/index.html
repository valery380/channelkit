Get a Stable Public URL

Use a free Cloudflare Tunnel to keep the same URL across restarts.

1
Create a Cloudflare account
Go to one.dash.cloudflare.com and sign up for free.
2
Add your domain to Cloudflare
Click Add a site, choose Free plan, update nameservers at your registrar.
3
Create a tunnel
Go to Zero Trust → Networks → Connectors, click Create a tunnel, choose Cloudflared.
4
Copy the tunnel token
Copy the long token after --token (starts with eyJh…).
5
Add a public hostname
Point it to localhost:4000 with HTTP service type.
6
Paste token & hostname here
Click Set up stable URL, paste, save, then Externalize.

The tunnel stays active as long as ChannelKit is running.
